JSON File Structure

Your JSON file must be an array of command objects with the following structure.

{
  "time": <number>,
  "command": "<string>",
  "value": "<string|number>"
}
  • time Time in seconds (s) from the start of the experiment. Must be non-negative and strictly increasing.
  • command The command to execute.
  • value The value associated with the command. Type depends on the command.
Commands Reference

The following commands are supported in the experiment configuration.

SET_CONTROL_MODE

Selects the control strategy of the electrolyzer.

AMP → Current mode (A) · WATT → Power mode (W)

SET_REGEN_MODE

Defines dryer regeneration behavior.

INSTANT → Start as soon as saturated · NEXT → Regenerate at next start

CHANGE_SETPOINT

Changes the operating setpoint.

Amp mode: 7.5–145 A · Watt mode: 500–14000 W
