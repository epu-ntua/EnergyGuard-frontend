JSON File Structure

Your JSON file must be an array of command objects with the following structure.

{
  "time": <number>,
  "command": "<string>",
  "value": "<string|number>"
}
  • time Time in seconds (s) from the start of the experiment. Must be a non-negative number and strictly increasing.
  • command The command to execute.
  • value The value associated with the command. Type depends on the command.
Commands Reference

The following commands are supported in the experiment configuration.

Command Description Allowed Values / Notes
set_control_mode Selects the control strategy of the electrolyzer. amp → Current mode (A)
watt → Power mode (W)
set_regen_mode Defines dryer regeneration behavior. instant → Start regeneration as soon as the column is saturated.
next → Regenerate the column at the next start.
change_setpoint Changes the operating setpoint. In amp mode: 7.5–145 A
In watt mode: 500–14000 W
