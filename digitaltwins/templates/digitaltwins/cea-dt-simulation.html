The DT Scenario Simulation is a CEA node that runs the SOEC (Solid Oxide Electrolysis Cell) Digital Twin on a scenario you provide, returning the raw time-series output of the simulated test.

Upload a scenario
One scenario file, a full SOEC simulation

Provide the operating conditions and degradation coefficients that define your test, either generated with the AI Scenario Generation node or written by hand. The Digital Twin runs the SOEC simulation and returns the raw output as an .h5 file, ready to download or open directly in JupyterHub.

How it works

Upload

Provide a generated or custom scenario file.

Run

Launch the SOEC Digital Twin simulation.

Monitor

Track simulation progress in real time.

Results

Download the resulting .h5 output file.

What you'll need to provide

Scenario file

A single JSON file, up to 10 MB, containing the time-series signals and scalar parameters for the test you want to simulate.

Time-series signals

9 variables — current, voltage, temperature, steam conversion and gas flowrates — each an array the same length as the time vector.

Scalar parameters

4 degradation coefficients, one per stack, applied as constant values across the whole simulation.

What is in the JSON file?

The scenario file contains time-series signals (varying over time) and scalar parameters (constant values) used to run the simulation.

Time-series signals

Arrays of values associated with the time vector. All signals must have the same length as timeStamp.

9 variables
Scalar parameters

Single values used across the entire simulation.

4 variables
Required Inputs Reference

Your JSON file must include the following variables.

Variable (Tag) Description Unit Valid Range
timeStamp Time vector of the scenario. s [0 : inf]
current Electrical current to apply on each stack of the electrolyser. A [-100 : 0]
voltageCell Mean cell voltage of the stacks to maintain (used only when a current is applied). V [1.285 : 1.3]
temperatureHotbox Temperature of the thermal enclosure of the stacks (used only when no current is applied). °C [700 : 800]
steamConversion Percentage of steam inlet flowrate converted into electricity by the system. – [0.4 : 1]
fuelElectrode_ratioH2O Percentage of steam in the gas mix at the inlet of the system. – [0 : 1]
fuelElectrode_flowrate Total flowrate at the inlet of the fuel side. NL/min [0 : 200]
oxygenElectrode_flowrate Total flowrate at the inlet of the air side. NL/min [0 : 200]
deltaP Pressure difference between the air and the fuel electrode of the cells inside the stacks. mBar [0 : 100]
coef_degradPerf[1] Coefficient to modify the initial electro-chemical performance of stack 1. – [0.5 : 1.5]
coef_degradPerf[2] Coefficient to modify the initial electro-chemical performance of stack 2. – [0.5 : 1.5]
coef_degradPerf[3] Coefficient to modify the initial electro-chemical performance of stack 3. – [0.5 : 1.5]
coef_degradPerf[4] Coefficient to modify the initial electro-chemical performance of stack 4. – [0.5 : 1.5]
